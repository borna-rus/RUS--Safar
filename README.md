# Smart Garage - Lab1

## Opis
Sustav pametne garaže implementiran na ESP32 koji demonstrira rad s višestrukim prekidima.

## Funkcionalnosti
- Upravljanje vratima (interrupt)
- Emergency STOP (najviši prioritet)
- Service mode
- Detekcija auta (HC-SR04)
- Automatsko zatvaranje vrata (timer)

## Prioriteti prekida
1. STOP (najviši)
2. Vrata
3. Service

## Komponente
- ESP32
- 3 tipke
- 5 LED
- HC-SR04
